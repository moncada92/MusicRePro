## Compile the project

```nodejs
  #Install dependencies (you must into folders API and frontend):
  npm install
```


```nodejs
  #Run project in mode production (You must into folder frontend):
  npm run build:prod
```

```nodejs
  #Run project in mode development:
  npm run build:dev-server
```
